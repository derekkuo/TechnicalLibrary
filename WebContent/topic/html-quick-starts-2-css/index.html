CSS基础知识

什么是CSS?
  • CSS 指层叠样式表（Cascading Style Sheets），层叠的意思是：多重样式定义被层叠为一。
  • CSS 是标准的布局语言，用来控制元素的尺寸、颜色、排版。
  • CSS 非常精确，功能强大，易于编写。
  • CSS 由 W3C 发明，用来取代基于表格的布局、框架以及其他非标准的表现方法。
  • 样式 (style) 定义如何显示 HTML 元素；样式通常存储于样式表中；外部样式表存储于 CSS 文件中。
  • 外部样式表 (External Style Sheet) 可以极大地提升你的工作效率，并且极大地节约服务器的带宽。
  • 纯 CSS 的布局与 XHTML 的结合，可使表现与结构分离，并使站点更易维护，易用性更好!

CSS的基础语法

好的！现在我们明白了CSS是什么，用来干什么的。下面我们将写一些简单的CSS样式控制.

CSS 语法由三部分构成：选择器、属性和值：

选择器找到需要被描述的HTML元素，给该元素指定的属性赋值. 我们可以这么理解:我们定义：在柜子里的书的封面是白色的.又或编号为1的学生的成绩是100。很明显：成绩是学生的属性，100是该属性的值，我们通过编号=1找到了这个学生,并把值赋给了这个学生。下面让我们来稍微分析一下CSS的三部分

1）选择器

作用：找到需要被描述的元素。当然这些元素可以是一个，也可以是很多个.

选择器分类：

派生选择器

通过依据元素在其位置的上下文关系来定义样式，你可以使标记更加简洁。这是什么意思呢？

简单的打个比方：柜子里的书。可以这么翻译。书是在柜子里的。这是的上下文关系就是：xxx里的xx.请看下面的例子:

Css:
	li strong {
		font-style: italic;
		font-weight: normal;
	}

对应的控制的HTML元素实例

HTML:
	<p><strong>我不在列表当中，样式对我不起作用</strong></p>
	<ol>
		<li><strong>strong元素在li元素内，样式控制有效</strong></li>
		<li>我没有样式控制</li>
	</ol>

上面 CSS文件描述的意思是：li元素中的 strong元素的样式是{。。}中定义的.现在我们先不去管那些详细的样式的值。我们接着往下看。

ID选择器

ID选择器可以为标有特定 id 的 HTML 元素指定特定的样式。这个选择器最容易理解的：我们假设有一群人，每个人都有一个特定的编号，我们如何去快速的找到某一个人呢？答案呼之欲出：我们按照号码找人就行了。请看下面的例子：

Css:
	#p1 {color:red;}
	#p2 {color:green;}

对应的控制的HTML元素实例


	<p id="p1">这个段落是红色。</p>
	<p id="p2">这个段落是绿色。</p>

类（class）选择器：

类选择器可以为标有固定class的HTML元素指定特定的样式。比如：人的耳朵是2只。人是一个类别，代表了所有的人，黄种人，白种人，黑人，美国人。。。等等。只要是人就被指定为有2只耳朵。请看下面的例子： 

CSS:	
	.center {text-align: center}

对应的控制的HTML元素实例

HTML：
	<h1 class="center">
	This heading will be center-aligned
	</h1>
	<p class="center">
		This paragraph will also be center-aligned.
	</p>

2）属性和值

属性是每个元素固有的东西,值和属性是固定的映射关系。比如人有一项属性身高，书有一项属性出版日期。这些属性是固有的。“人的出版日期是2011-5-3”又或“书的身高是2米”—这显然是不太正常的思维。

以下列举一下常见的元素的常见的属性：

margin:外边距
  • margin-left:
  • margin-right:
  • margin-top:
  • margin-bottom:
  • margin:0px;
padding:内边距
  • padding-left;
  • padding-right;
  • padding-top;
  • padding-bottom;
边框：
  • border:宽度，样式，颜色
  • border-left:左边框
  • border-right：右边框
  • border-buttom:下边框
  • border-top:上边框
背景:
  • background-color:#cccccc
  • background-image:url(图片的路径)
  • background-repeat:repeat-x/repeat-y/no-repeat;
  • background-attchment:fixed/scroll(默认);
  • background-position
文字：
  • font-size 文字大小
  • font-weight：粗细 100-900
  • font-family：字体
  • text-decoration：none/underline 下划线
  • cursor:pointer/wait 鼠标形状
  • text-align:left/center/right 文字位置
列表：
  • list-style-type:none; 列表前面的符号去掉,如去掉小圆点
关键属性：
  • float:浮动 left/right
  • clear:取消浮动 both;
  • display: 显示方式
  • block:HTML标记以快标记的形式来显示
  • inline:html标记以行内标记来显示
  • none:不显示
  • position属性 移动图片,将图片针对父标记的位置进行偏移
  • static 默认值，表示把标记按照从上到下，从左到右进行摆放
  • absolute 相对父标记进行偏移
  • relative 先按照浏览器的默认方式摆放，再在摆放的位置上进行偏移
