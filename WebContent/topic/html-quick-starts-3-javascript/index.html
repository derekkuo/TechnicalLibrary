JavaScript基础知识

什么是JavaScript？javaScript干什么？

JavaScript是一种基于对象和事件驱动并具有相对安全性的客户端脚本语言。同时也是一种广泛用于客户端Web开发的脚本语言.以下简称JS。

JS主要是用来完成一下功能：

  • 前端验证（就是对用户提交的表单中的数据进行验证，比如验证用户名是否为空,如果验证通不过，则浏览器不会将数据提交给服务器）。
  • ajax核心技术之一，用于异步向服务器发送请求并动态更新页面。
  • 与页面进行交互，生成动态的效果。
  • 获得浏览器相关的信息

JS在哪里执行？JS可以写在哪里？JQuery和propertype是什么？

js程序在浏览器端执行,可以写在.js结尾的文件里或者HTML，jsp的页面头中。

Jquery和Propertype是JS的框架。这里的框架一词大家不要想得太夸张，我们理解为很多用JS写的方法的集合体（JS文件）即可，你也可以轻松的写一个JS框架的，只不过可靠性似乎…

JavaScript文件的基本结构:

JavaScript是采用事件调用机制进行触发的。每个函数的都有自己的函数名称。返回类型，函数体。我们来看看下面这个函数：


		function  sayHello{
			alert（”hello”）；
		}

当某个事件调用sayHello这个函数的时候，这个函数会说：“hello”.当然，我们还可以做一些别的事情。比如说对页面的某些东西进行更改。

JavaScript的简单示例：

看了这么多的基础知识，相信大家都比较烦躁了。不着急，我们马上就开始我们的第一个应用。在此之间，我们必须解释一下什么是事件调用。在我们以前的HTML文件中，某些元素有一些属性：比如onclick（）这个表示被该组件单击时采取什么行动。所以：事件调用是由于某个事件而引发一系列操作的过程。我们看下面的这个例子：

		
		
			<html>
				<head>
					<script>
						function f4(){
							var str1 = "123";
							var str2 = "123";
							var str3 = 123;
							if(str1 == str2){
								alert('str1等于str2');
							}
							if(str1 === str3){
								alert('str1等于str3');
							}
							var s1 = null;
							var s2;
							alert(s1 == s2);
						}
					</script>
				</head>
				<body style="font-size:30pt;">
					<a href="#" onclick="f4();">test</a>
				</body>
			</html>

在上面的例子中，我们定义了一个JS函数f4，我们在HTML页面元素中的一个连接定义了函数调用：

		
			
				<a href="#" onclick="f4();">test</a>

当这个连接的被单击的时候，产生了一个onclick（）事件。按照上面元素中的声明，产生的这个事件将调用f4函数。所以f4函数体中的操作将被执行。

某些同学比较细心，注意到了”onclick="f4();"”中的f4();为什么有个“；”?这里不得不说，一个事件可以调用多个函数。多个函数之间用“；”分隔开来。如下例：

		
			
				<html>
					<head>
						<script>
							function f1(){
								alert("1");
							}	
							function f2(){
								alert("2");
							}
						</script>
			
					</head>
					<body style="font-size:30px;">
						<a href="#" onclick="f1();f2();">test</a>
					</body>
				</html>

单击将弹出1,2的提示框。

某些同学又想，函数与函数之间可以互相调用吗？答案是肯定的。如下例：

		
			
				<html>
					<head>
						<script>
							function f1(){
								alert("1");
								f2();
							}	
							function f2(){
								alert("2");
								f1();
							}
							</script>
					</head>
					<body style="font-size:30px;">
						<a href="#" onclick="f2();">test</a>
					</body>
				</html>

这个例子则实现了函数之间的相互调用，用一次上面的例子，相信一定会对JS函数之间的相互调用记忆深刻。
