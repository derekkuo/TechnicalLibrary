HTML基础知识

1) 什么是HTML
  • HTML 指超文本标签语言。
  • HTML 文件是包含一些标签的文本文件。 这些标签告诉 WEB 浏览器如何 显示页面。
  • HTML 文件必须使用 htm 或者 html 作为文件扩展名。
  • HTML 文件可以通过简单的文本编辑器来创建.

2) HTML文件的基本结构

首先，我们要明白HTML的<..>的意思：在HTML中,每行<..>开头的我们称之为 标签。 这些标签是固定写死的，并且有着自己独特的含义。 比如：<!—注释内容 -->表示这是一个注释。

一个HTML文件的结构如下所示：


<!--表示这是一个HTML文件，并且有开始的含义-->
<html>
<head>
	<title>标题</title>
	<!--用于模拟http消息头-->
	<!--比如，以下消息头的作用，是告诉浏览器，返回的是一个html，
	并且，应该以utf-8的编码来显示该页面,html文件保存的编码应该与meta声明的编码一致-->
	<meta http-equiv="content-type"content="text/html;charset=#">
	<!--引入脚本，定义行为-->
	<script>
	</script>
	<!--定义样式-->
	<style>
	</style>
	<!--引入外部样式-->
	<link>
</head>
<body>
	<!--页面显示的数据,我们需要写的内容-->
</body>
</html>

通过对以上HTML文件结构的观察，我们发现有如下的特点：

  • 标签的出现时成对出现的。有<..>表示开始，一般都有</..>表示结束.我们可以形象的称<..>为标签头，</..>为标签尾。
  • 某些标记没有尾.HTML的语法要求是比较宽松的。

好的。HTML的文件结构就是这样。可是为什么是这样呢？ 我们往下看。

3) web开发标准

什么是w3c?

中文意思是W3C理事会或万维网联盟。W3C组织是对网络标准制定的一个非赢利组织，像HTML、XHTML、CSS、XML的标准就是由W3C来定制。

上面展示了HTML文件的结构。这样的结构是有一定的设计规定的。w3c建议，开发一个页面，应该这样设计：

  • 页面的结构（包括了数据），由html文档来负责。就是各种标签。
  • 页面的外观，由css文档来负责。
  • 页面的行为，由javascript文档来负责。

以上三条我们可以形象的理解为：一个html文档是一台有了结构的机器。这台机器的组成零件是各种各样的标签。可能这台机器不太好看，于是 我们想给它进行装饰一番，于是我们用到CSS文档。现在这台机器好看 了，但是不够灵活，死气沉沉的，于是我们用java script文档来预设 一些指令，让它能够听我们的话，完成我们指派的任务。

HTML常用的标签

好的，现在，我们对HTML的基本结构和起源有了一定的了解，接下来，我们开始学习常用的一些标签。这些标签对于我们非专业美工的开发人员，足以满足工作的需要。

1) 链接

a. 基本使用

 

<a href="url地址" target="_blank">描述</a>
  • _blank:在新的窗口中打开url地址所对应的内容
  • _self: 在当前窗口打开（默认）

b. 使用图片作为链接

 

<a href="url地址"><img src="图片地址" width="" 
height="" border=""/></a>

c. 使用热点

 

<img src="index04.jpg" width="500" height="500" border="0" usemap="#m1" />
<map name="m1" id="m1">
	<area shape="rect" coords="…" href="#" />
	<area shape="circle" coords="…" href="#" />
	<area shape="poly" coords="…" href="#" />
</map>

当shape="rect"（矩形）时，coords的值为左上角和右下角的坐标值; 当shape="poly"（多边形）时，为各顶点值; 当shape="circle"（圆形）时，为圆心坐标值和半径值。coords值依照图片大小的不同和所希望链接区域的不同而有所不同

d. 使用锚点:在同一个页面内部跳转

 

<a name="top"></a>
<a href="#top"></a>

e. 发送邮件

 

<a href="mailto:sally@126.com?subject=hello">
</a>

f. 链接中的伪样式（css中再讲）

3) 表格

a. 基本使用

 
	
<table border="" cellpadding="" cellspacing="" width="">
	<tr>
		<td align="">id</td><td>name</td><td>salary</td>
	</tr>
	<tr>
		<td>1</td><td>zs</td><td>2000</td>
	</tr>
</table>
  • border:边框的宽度
  • cellpadding:单元格里面的数据与单元格边框之间的距离
  • cellspacing:单元格之间的距离
  • width:60%/600 宽度
  • align: 水平位置 left/center/right
  • valign:垂直位置 top/middle/bottom

b. 不规则的表格
  • cospan:跨列合并
  • rowspan:跨行合并

c. 表格的另一种写法

 
<table>
	<caption>标题</caption>
	<thead></thead>
	<tfoot></tfoot>
	<tbody></tbody>
	<tbody></tbody>
</table>
  • thead:表头，0/1
  • tfoot:表脚,0/1
  • tbody:表体,1/n

d. 表格可以嵌套

4) 表单

什么是表单：用于收集用户的信息，比如 登录表单可以让用户输入登录的用户名、密码， 这些信息会提交给服务器来验证。

<form>
	<input type="text"/>
</form>

input标记:

  • type="text": 文本输入框
  • type="submit":提交按钮
  • type="reset":重置按钮
  • type="password":密码输入框
  • type="radio":单选
  • type="checkbox":多选
  • type="file":上传文件(必须给form添加 enctype="multipart/form-data")
  • type="hidden":隐藏域

非input标记:

  • <textarea>:多行文本输入框
  • <select>:下拉列表/多选框

5) 框架

什么是框架，用来将一个窗口划分成多个子窗口。

a. <frameset>,<frame>

注意：frameset不能写在<body>里,可以嵌套使用。

b. <iframe>

可以使用在<body>里，表示嵌入一个子窗口。
